# 多模态情感分类实验（AI_lab5）
本仓库是多模态情感分类实验的代码实现，包含文本-图像融合的3种模型。

## 环境依赖
- Python 3.8+
- torch==1.12.0
- transformers==4.24.0
- pandas==1.5.3
- scikit-learn==1.2.2
- pillow==9.3.0

## 运行步骤
1. 克隆仓库：`git clone https://github.com/MinDegeI/AI_lab5.git`
2. 安装依赖：`pip install -r requirements.txt`（可新建`requirements.txt`存放依赖）
3. 运行实验：`python main.py`
4. 生成报告：`python history_report.py`
